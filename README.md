<h4>What is Procolab?</h4>
<li>
  Procolab is the perfect tool for collaborating in real-time with team on code
  development and project management. With its intuitive interface and powerful
  collaboration features, Procolab is the perfect tool for coding projects.
</li>

<h4>Features</h4>
<ul>
  <li>1. Real-time collaboration</li>
  <li>2. Code editor</li>
  <li>3. Project management</li>
  <li>4. Chat</li>
  <li>5. File saving and sharing</li>
  <li>6. Code execution</li>
  <li>7. Code sharing</li>
</ul>

<h4>How to use?</h4>
<ul>
  <li>
    Get Started by selecting "Sign in Now" on the home page.
     <ul>
        <img
  style="display: inline" src="https://user-images.githubusercontent.com/59359937/194952072-edfeb56b-9b1b-4e39-9b4c-541686a28659.jpg"
  width="500"
/>
     </ul>
     <ul>
      <li>
        Login with an existing account.
        <ul>
          <img style="display: inline" src="https://user-images.githubusercontent.com/59359937/194952741-0c69dbc7-0082-4442-96a6-142c37bb2fb5.jpg" width="300"/>
          <li>Enter your registered email address</li>
          <li>Enter the password you used while signing up</li>
          <li>Click on continue.</li>
          <li>You will be successfully logged in</li>
        </ul>
      </li>
      <li>
        Create a new account.
        <ul>
          <img style="display: inline" src="https://user-images.githubusercontent.com/59359937/194953139-703069fa-a140-42a5-86d4-09b0fe50a574.jpg" width="300"/>
          <li>Click on Sign Up</li>
          <li>Enter an email you want your account to be associated with</li>
          <li>Enter a password and confirm it.</li>
            <li>Click on continue.</li>
          <li>You will be successfully registered and logged in</li>
        </ul>
      </li>
      <li>
        Login with a Google account.
        <ul>
          <img style="display: inline" src="https://user-images.githubusercontent.com/59359937/194953250-d82b1b49-831a-4f80-af0e-d9958f6d4e04.jpg" width="300"/>
          <li>Click on Continue with Google</li>
          <li>Select the Gmail account.</li>
          <li>You will be successfully logged in.</li>
        </ul>
      </li>
    </ul>
  </li>
  <br>
  <li>
    Create a new project by selecting "Make a Project"
    <ul>
      <img style="display: inline" src="https://user-images.githubusercontent.com/59359937/194953388-d5b1af9e-377a-4b82-8107-4918a2449ccb.jpg" width="300"/>
    </ul>
    <ul>
      <img style="display: inline" src="https://user-images.githubusercontent.com/59359937/194954072-05f38b5c-5145-4984-b7ca-953e3570ebe8.jpg" width="300"/>
      <li>Enter a name for the project</li>
      <li>Select coding language</li>
      <li>Give your project a brief description</li>
      <li>Select upto 5 Collaborators for the project</li>
      <li>
        How to add a Collaborator?
        <ul>
          <img style="display: inline" src="https://user-images.githubusercontent.com/59359937/194954196-f459b873-7d9f-4f6f-a67f-bbf8662dae67.jpg" width="300"/>
          <li>Click on the "Add Collaborators" button</li>
          <li>Enter the email of the Collaborator</li>
          <li>Select the access permissions for the collaborator</li>
          <li>Click on the "Add" button</li>
        </ul>
      </li>
      <img style="display: inline" src="https://user-images.githubusercontent.com/59359937/194954388-47da755d-7e85-4a63-951d-727f91561f86.jpg" width="300"/>
      <li>Click on the "Create" button</li>
      <li>A new project is created in the database and the collaboration interface will be displayed</li>
    </ul>
  </li>
  <br>
  <li>
    Manage your projects by selecting "All Projects"
     <ul>
         <img style="display: inline" src="https://user-images.githubusercontent.com/59359937/194954777-49eb641b-d4c0-4f55-985b-ffbc379cdf02.jpg" width="300"/>
     </ul>
    <ul>
      <li>If there are no projects, a "No projects" screen is shown. <br> Go back to homepage to create a project</li>
         <img style="display: inline" src="https://user-images.githubusercontent.com/59359937/194954969-8a8ed4d4-ac48-452f-acfe-a156024318b8.jpg" width="300"/>
      <li>If there are project created by you or you' re added as a <br> collaborator, the following project management interface is shown.</li>
         <img style="display: inline" src="https://user-images.githubusercontent.com/59359937/194955047-416a25be-1c0b-4a8f-9485-ae2e09e404b5.jpg" width="300"/>
      <li>
        The project management interface consists of the following columns:
        <ul>
          <li>Project Name</li>
          <li>Project Description</li>
          <li>Project Language</li>
          <li>Project Collaborators</li>
          <li>Date the project was Created</li>
          <li>Permissions you have in the project</li>
          <li>If you're an admin or not</li>
          <li>Project Actions - View, Edit & Delete</li>
        </ul>
      </li>
      <li>You can perform the following operations on each columns:
        <ul>
          <img style="display: inline" src="https://user-images.githubusercontent.com/59359937/194955173-790c716e-9c32-4eca-8c01-1fa5de1f4f31.jpg" width="300"/>
          <li>Sort in Ascending Order</li>
          <li>Sort in Descending Order</li>
          <li>Show/ Hide the column</li>
          <li>Put filters to show only selected entries</li>
        </ul>
      </li>
      <li>There are other options like settings, search, transfer data, etc too.</li>
      <img style="display: inline" src="https://user-images.githubusercontent.com/59359937/194955300-14660b95-d836-4eb3-b0e4-dc73c78c2329.jpg" width="300"/>
    </ul>
  </li>

   Collaborate with your team on a project from the collaboration interface

            <img style="display: inline" src="https://user-images.githubusercontent.com/59359937/194955394-edcedf37-dbcb-41f7-a112-90423096eec6.jpg" width="300"/>
         </ul>
        <ul>
            <li>Has specific language support</li>
            <li>Supports syntax highlighting</li>
            <li>Supports code completion</li>

      </li>

        <ul>

            <li>Lists all the collaborators on the current project</li>
            <li>Shows the active status of each collaborator</li>
            <li>Shows the permission each collaborator has</li>
        </ul>
      </li>
     </ul>
